# AlienVault Appliance SOAR Integration

This document details the tools provided by the AlienVault Appliance SOAR integration.

## Tools

### `alien_vault_appliance_get_pcap_files_for_events`

Get PCAP files for events in an alert

**Parameters:**

*   `case_id` (str, required): The ID of the case.
*   `alert_group_identifiers` (List[str], required): Identifiers for the alert groups.
*   `target_entities` (List[TargetEntity], optional, default=[]): Optional list of specific target entities (Identifier, EntityType) to run the action on.
*   `scope` (str, optional, default="All entities"): Defines the scope for the action.

### `alien_vault_appliance_enrich_assets`

### `alien_vault_appliance_fetch_last_pcap_files`

Fetch last PCAP files from AlienVault

**Parameters:**

*   `case_id` (str, required): The ID of the case.
*   `alert_group_identifiers` (List[str], required): Identifiers for the alert groups.
*   `number_of_files_to_fetch` (str, required): e.g. 10
*   `target_entities` (List[TargetEntity], optional, default=[]): Optional list of specific target entities (Identifier, EntityType) to run the action on.
*   `scope` (str, optional, default="All entities"): Defines the scope for the action.
